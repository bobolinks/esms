# esms
Esms is a http server for esm project.

# Installation:

## Globally install
  npm i -g esms

## Locally install
  npm i esms

# Usage:
  esms [path] [options]

## Available Options:
`--port` Port to use (defaults to 80). 
`--config` Config file(name like `esms.config.ts`) to be used to run serve.
